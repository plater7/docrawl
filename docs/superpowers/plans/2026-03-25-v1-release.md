# DocRawl v1.0.0-public Release Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring DocRawl from v0.10.0 to v1.0.0-public by completing all P0–P3 checklist items.

**Architecture:** FastAPI service (Python 3.12/3.13) with multi-provider LLM integration (Ollama, OpenRouter, OpenCode, LM Studio). Pipeline: URL discovery → LLM filter → 5-level scrape fallback → LLM cleanup → markdown/JSON output. Deployed as Docker container with SSE event stream and industrial web UI.

**Tech Stack:** Python 3.12/3.13, FastAPI, pytest (asyncio_mode=auto), GitHub Actions (14 workflows), Docker/GHCR, Playwright, httpx, slowapi (rate limiting), sse-starlette

### P1.4 — Add doc-freshness CI check

**File to create:** `scripts/check_doc_freshness.py`

- [ ] Create script:
  ```python
  #!/usr/bin/env python3
  """Verify API_VERSION in src/main.py matches docs/PROJECT_STATUS.md header.
  Exit 0 = fresh. Exit 1 = stale.
  """
  import ast
  import re
  import sys
  from pathlib import Path

  ROOT = Path(__file__).parent.parent

  tree = ast.parse((ROOT / "src" / "main.py").read_text())
  api_version = None
  for node in ast.walk(tree):
      if isinstance(node, ast.Assign):
          for t in node.targets:
              if isinstance(t, ast.Name) and t.id == "API_VERSION":
                  api_version = ast.literal_eval(node.value)

  if not api_version:
      print("ERROR: Could not find API_VERSION in src/main.py")
      sys.exit(1)

  status_text = (ROOT / "docs" / "PROJECT_STATUS.md").read_text()
  m = re.search(r"DocRawl v(\d+\.\d+[\.\d]*)", status_text)
  if not m:
      print("ERROR: Could not find version in docs/PROJECT_STATUS.md")
      sys.exit(1)
  doc_version = m.group(1)

  if api_version != doc_version:
      print(f"STALE: main.py API_VERSION={api_version!r} but PROJECT_STATUS.md says v{doc_version}")
      sys.exit(1)

  print(f"OK: API_VERSION={api_version!r} matches PROJECT_STATUS.md")
  ```
